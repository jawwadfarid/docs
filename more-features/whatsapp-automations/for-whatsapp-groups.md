### Privacy & Data Handling

#### Key Principle: Ephemeral by Default

Your Automation runs in short-lived sandboxes ‚Äî fresh for every event and deleted right after.\
Each run lasts 2‚Äì30 seconds, then the environment is destroyed.

* Created new for each event
* Deleted after execution
* No local storage between runs
* No automatic data retention

**Meaning:**

* Your bot forgets everything after each run.
* Data is only kept if you _explicitly choose_ to store it.
* This ensures privacy-first design ‚Äî no accidental data retention.
