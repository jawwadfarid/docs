# WhatsApp automations

We believe you should have access to powerful automations straight from your phone.&#x20;

That's why we've launched WhatsApp automations so you can easily call, trigger, and schedule workflows between CodeWords and your mobile.&#x20;

There are two main types of WhatsApp automations that we'll go through:&#x20;

* [**WhatsApp automations for groups**](https://docs.codewords.ai/~/revisions/TEXQfTErtnrNKxMHQA6A/more-features/whatsapp-automations/for-whatsapp-groups)&#x20;
* [**WhatsApp automations for DMs**](https://docs.codewords.ai/~/revisions/TEXQfTErtnrNKxMHQA6A/more-features/whatsapp-automations/for-whatsapp-dms)

Both options include a range of functionality so you can create intelligent systems that work where you live.&#x20;
